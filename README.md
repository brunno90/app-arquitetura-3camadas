# app-arquitetura-3camadas

Para executar a aplicação:

1 - Utilize o servidor de aplicação JBoss para subir o arquivo war, ou execute os fontes utilizando uma IDE

2 - Instale o banco de dados desejado.

3 - Crie uma base com o nome "trabalho".

4 - Crie um jndi no Servidor de aplicação com o id "jboss/jdbc/trabalho" apontando para o base criada.

5 - Inicie o servidor.
